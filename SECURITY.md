# Security Policy

## Supported Versions

Currently only the `master` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of AirVision seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, please send an email to the maintainers (security@airvision.example.com - placeholder).

### Our Process

1. **Acknowledgment**: We will acknowledge your report within 48 hours.
2. **Investigation**: We will investigate the issue and determine its impact.
3. **Fix**: If confirmed, we will work on a patch.
4. **Disclosure**: Once fixed, we will provide a public disclosure of the vulnerability.

Thank you for helping keep AirVision secure!
